# iot-Server

install php / mysql
```
sudo apt-get install php mysql-server php-mysql
```

clone project
```
git clone https://github.com/iot-server/iot-server.git
```

install

```
cd iot-server
composer install
```
start
```
cd public
php -S localhost:3000
```

api
```
http://localhost:3000/api
```
